github+hexo搭建个人博客

发表于 2018-03-26

声明：本文为参照ryanluoxu原创文章

github 准备工作

1、注册github 创建一个以：http://***.github.ol结尾的Repository（项目的名称最好和github账户名一样）
Repository name:honghong0626.github.io勾选Initialize this repository with a REAMECrate repository( README里面的内容随便编写)
2、在项目的右上角有settings 进去 往下有一个 GitHub Pages

Source (选中master branch —save)会自动生成一个连接 这就是之后项目打开的链接

安装node git

官网均可以下下载
node -v git -v 查询版本

安装hexo

npm install -g hexo-cli 回车开始安装 安装完了之后可以 hexo -v得到hexo-cli:1.0.4…等一系列的数据

创建本地博客

我是c盘 （你们自己决定） 创建文件夹blog 鼠标右键Git Bash Here 可以看打开的命令窗口显示的是当前文件夹的位置

输入
hexo init 初始化blog
npm install 安装依赖包
hexo g 生成网页（generate）
hexo s 生成的网页放在了本地服务器上

在浏览器中输入 http://loaclhost:4000/就可以看到成果 显示的一篇日志 hello world

Git Bash 中 Ctrl+C 是结束命令

新建日志：
hexo new “My first post” 在blog/source_posts下面就会有一个my-first-post.md的文件 编辑这个文件 回到保存

hexo g hexo s看浏览器就会看到新建的日志

将本地 Hexo 博客部署在 Github 上前面两个部分，我们已经有了本地博客，和一个能托管这些资料的线上仓库。只要把本地博客部署（deploy）在我们的 Github 对应的 Repository 就可以了

1:获取github对应的Repository的链接
登陆github: 进入到hongl.github.io
点击clone or download
复制url待用
2:修改博客的配置文件
打开配置文件blog/_config.yml
找到#Deployment 填入一下内容：
deploy:
type: git
repository: 复制的url
branch: master 

保存退出

使用NEXT主题

更多的HEXO的主题看这里

操作如下：
1:Git Bash 输入：$ git clone https://github.com/iissnan/hexo-theme-next themes/next

2:修改博客配置文件

打开blog/_config.yml
找到 theme:
把 Hexo 默认的lanscape 修改成next 即：theme:next
找到 # Site 添加博客名称 作者名字等
在language后面填入 en或者zh-Hans 选择英文或者中文
找得到 #URL 填入url

3:重新生成部署即可

回到Git Bash 输入 hexo g -d 就好了
先把修改的内容生成网页 在部署
